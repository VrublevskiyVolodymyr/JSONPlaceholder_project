About Project

This project is a demonstration of using the JSONPlaceholder API to fetch and display data about users and posts. It includes several pages:

  • Home Page: The main landing page of the project.
  • Users: Lists all users with links to their details.
  • User Details: Displays detailed information about a selected user.
  • Post Details: Shows detailed information about a selected post, including its comments.
  • User Posts: Displays a paginated list of posts for a selected user.
  • About: Provides information about the project.

The project includes a header with a theme switcher button, and it is fully responsive for mobile devices. The User Posts page also features pagination for navigating through the posts.
